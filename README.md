# 334-screensaver

Documents and Code to create classic screen savers using pygame.

Works on Linux, Raspberry Pi OS, Windows and MacOS

Samples for:

- 2D Pipes
- Starfield
- Flying Toasters
- Stars, Starships, Tosters and Camels
- 8bit Aquarium
- Bouncing Logo
- The Matrix
- Flying Plane
- Rotating 3D Cube
- 3D Pipes
- One Ring
- Floating Astronaut
